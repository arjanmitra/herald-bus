This is a tool that allows a user to upload PDF files and extract the data using Herald APIs. Users can upload files directly to Herald's servers, call extraction APIs, and download the extracted data to Excel. Users can also track all files extracted directly in the tool.

## Getting Started

First, download all dependencies:

```bash
npm install
```

Then,run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
